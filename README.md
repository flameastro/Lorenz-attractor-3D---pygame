## 👾 Execute

### Install Dependencies
```bash
pip install pygame
```

[if your python version is 3.12+, try:](https://github.com/pygame/pygame/issues/4712)

```bash
python -m pip install pygame-ce
```

```bash
python main.py
```

if you get any error, try:

```bash
python3 main.py
```
